# Results Directory

This folder is intended to hold the output files generated by the simulation scenarios.

---

## Overview

Each scenario script automatically creates its own results folder in the root of the repository when it finishes running. The results are **not stored in this directory** during simulation — this folder exists as a placeholder and reference.

---

## Output Folder Structure

Each scenario produces a folder named:

```text
Results_Combined_<Scenario_Name>/
```

For example:

```text
Results_Combined_Scenario1_Baseline_BNM/
Results_Combined_Scenario2_DER_Heavy_BNM/
Results_Combined_Scenario3_Bulk_Integrated_BNM/
Results_Combined_Scenario4_Bulk_Extreme_BNM/
Results_Combined_Scenario5_Modern_Grid_BNM/
Results_Combined_Scenario6_Bulk_Collapse_BNM/
```

---

## Files Inside Each Results Folder

Each results folder contains two files:

### 1. Checkpoint File

```text
Checkpoint_FULL.mat
```

This is saved automatically after every completed Monte Carlo iteration. If the simulation is interrupted, it can be resumed from the last completed iteration by re-running the scenario script — it will detect the checkpoint and continue from where it left off.

### 2. Final Excel Output

```text
Final_Results_<Scenario_Name>.xlsx
```

This is the main results file exported at the end of the simulation. It contains the following sheets:

| Sheet | Contents |
|---|---|
| `System_Totals` | Total EENS, load shedding, line overload, and wind curtailment per Monte Carlo iteration for all four rating cases |
| `Bus_Shed_SLR` | Bus-level load shedding per iteration under Static Line Rating |
| `Bus_Shed_P5` | Bus-level load shedding per iteration under PLR 5% |
| `Bus_Shed_P10` | Bus-level load shedding per iteration under PLR 10% |
| `Bus_Shed_P15` | Bus-level load shedding per iteration under PLR 15% |
| `Line_Over_SLR` | Branch-level overload severity per iteration under Static Line Rating |
| `Line_Over_P5` | Branch-level overload severity per iteration under PLR 5% |
| `Line_Over_P10` | Branch-level overload severity per iteration under PLR 10% |
| `Line_Over_P15` | Branch-level overload severity per iteration under PLR 15% |
| `Farm_Curt_SLR` | Wind farm curtailment per iteration under Static Line Rating |
| `Farm_Curt_P5` | Wind farm curtailment per iteration under PLR 5% |
| `Farm_Curt_P10` | Wind farm curtailment per iteration under PLR 10% |
| `Farm_Curt_P15` | Wind farm curtailment per iteration under PLR 15% |

---

## Rating Cases

All results are tracked across four line rating cases for direct comparison:

| Label | Description |
|---|---|
| `SLR` | Static Line Rating — fixed conservative benchmark at 0.01% exceedance |
| `PLR 5%` | Probabilistic Line Rating at 5% exceedance |
| `PLR 10%` | Probabilistic Line Rating at 10% exceedance |
| `PLR 15%` | Probabilistic Line Rating at 15% exceedance |

---

## Note on Repository Storage

Large result files and checkpoint `.mat` files are excluded from the repository via `.gitignore`. To reproduce the results, the scenario scripts must be run locally after generating the required input files.
